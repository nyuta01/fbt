# Incident Response Runbook Format

The output must use this exact section order:

## Purpose

## Scope

## Detection

## Severity Classification

## Immediate Response

## Investigation

## Mitigation

## Recovery

## Customer Communication

## Escalation

## Post-Incident Follow-up

## Maintenance Notes

## Source Evidence

Each procedure section must use numbered steps. `Source Evidence` must list the
input files or incident IDs used for each major procedure.
