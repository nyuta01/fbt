# Support Resolution Manual Format

The output must use this exact section order:

## Audience

## When to Use

## Intake Checklist

## Triage

## Resolution Procedure

## Escalation

## Customer Response Templates

## Agent Notes

## Maintenance Notes

## Source Evidence

Each `Resolution Procedure` must use numbered steps. `Customer Response
Templates` must be safe to paste into customer-facing channels after replacing
bracketed placeholders.
