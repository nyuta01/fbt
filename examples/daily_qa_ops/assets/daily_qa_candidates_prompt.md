# Task

Read today's customer questions, support answers, and product docs.

Produce three separate candidate files:

1. FAQ candidates for questions that have a clear reusable answer.
2. Manual patch candidates for answers that should update official docs.
3. Unresolved questions when evidence is missing or contradictory.

Do not invent product behavior. If the answer is not supported by the source
files, put it in unresolved questions.
