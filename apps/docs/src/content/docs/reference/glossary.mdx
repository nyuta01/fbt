---
title: Glossary
description: Short definitions of common fbt terms.
---

| Term | Meaning |
|---|---|
| Artifact | Logical output managed by fbt |
| Artifact version | Immutable content snapshot with a digest |
| Eval | Deterministic or human quality gate |
| Manifest | Parsed project graph and metadata |
| Output candidate | Runner-produced file or directory before commit |
| Policy | Read, write, network, cost, and review constraints |
| Runner | External process that executes transform logic |
| Source | Read-only external input file, glob, or directory |
| Transform | Contract that creates artifacts from sources or upstream artifacts |
