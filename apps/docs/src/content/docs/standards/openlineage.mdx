---
title: OpenLineage export
description: Export fbt runs as OpenLineage RunEvent NDJSON.
---

`fbt export openlineage` emits OpenLineage RunEvent NDJSON.

```sh
fbt export openlineage \
  --project-dir /tmp/fbt-lineage-demo \
  --output /tmp/fbt-lineage-demo/target/lineage/openlineage.ndjson
```

The command prints a short summary:

```text
Export: openlineage
  Format  OpenLineage RunEvent NDJSON
  Output  /tmp/fbt-lineage-demo/target/lineage/openlineage.ndjson
  Events  2
  Next    post or ingest this NDJSON with a Marquez/OpenMetadata-compatible workflow
```

The export includes fbt-specific facets for artifact versions and lineage
context while keeping raw source content out of the payload.

## Where to send it

Use OpenLineage-compatible tools such as Marquez directly, or route the events
through an OpenLineage ingestion path into OpenMetadata.

For a running Marquez endpoint, the opt-in smoke target posts generated fbt
events:

```sh
FBT_MARQUEZ_URL=http://localhost:5000 make standard-backend-smoke
```

fbt core does not add a direct OpenMetadata publisher because that would require
server credentials, entity upsert semantics, and an OpenMetadata-specific SDK or
API dependency.
